# armRover
Arm rover a prototype rover which comes integrated with a robot arm which is controlled through computer vision using gestures.

## Authors

- [@jorgeeluisvg](https://www.github.com/jorgeeluisvg)

## Materials 

```bash 
  -2 Arduino Uno
  -2 Breadboards
  - LD293 
  -5 MG995 Servomotors
```

## Requirements 
python 3.6
cvzone == 1.5.2
pyserial == 3.5
